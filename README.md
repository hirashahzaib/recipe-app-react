# 🍽️ Recipe Finder App

A React-based web app that allows users to search and explore recipes from different countries using an external API.

## 🚀 Features

* 🔍 Search meals by name
* 🌍 Filter meals by country (Canadian, Indian, etc.)
* 📖 View full recipe with ingredients
* 🖼️ Interactive UI with modal popup
* ⚡ Fast performance using Vite

## 🛠️ Tech Stack

* React (Hooks)
* Vite
* Bootstrap
* TheMealDB API


## 🌐 Live Demo

http://recipe-meal-app-react.vercel.app

## 📂 Installation

Clone the repository:
git clone https://github.com/hirashahzaib/recipe-app-react.git

Go to project folder:
cd recipe-app-react

Install dependencies:
npm install

Run the project:
npm run dev
